# Kompas - aplikacja na Androida / Compass - Android app

(PL) Aplikacja kompasu na system operacyjny Android na telefon napisana w języku Kotlin. Praktycznie cały kod aplikacji zawarty jest w pliku MainActivity.kt. Aplikacja jest prosta, nie posiada żadnych dodatkowych funkcji za wyjątkiem możliwością zmianu motymu wyglądu aplikacji (jasny, ciemny). Moim celem nadrzędnym przy tworzeniu tej aplikacji była jej prostota oraz schludny wygląd dopasowywany do osobistych preferencji.

(EN) A compass app for Android, written in Kotlin. Virtually all of the app's code is contained in the MainActivity.kt file. The app is simple, with no additional features except for the ability to change the app's theme (light or dark). My primary goal in creating this app was simplicity and a clean look that could be customized to suit personal preferences.
